# Job-Market-Analysis-using-SQL-and-PowerBI

This project focuses on analyzing **job market trends** for data science-related positions using **SQL** and **Power BI**. The project includes database analysis, salary trend analysis, company hiring trends, skill demand analysis, education impact, and interactive dashboard visualization to derive meaningful insights from job market data.

## Objective

* Analyze job market trends using SQL queries and Power BI visualizations.
* Identify top hiring industries, companies, and required skills.
